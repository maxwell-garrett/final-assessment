To run code:

1. you must download the GSS dataset through http://www.chass.utoronto.ca/, select General Social Survey 2017. Follow specific instructions in beginning of gss_cleaning.R, save to data directory.

2. Download https://www.elections.ca/res/rep/off/ovr2019app/51/data_donnees/table_tableau08.csv, this is the csv containing the result from the election. Save to data directory

3. first you must run gss_cleaning.R to produce gss.csv

4. then you can run finalproject.rmd to reproduce results.
